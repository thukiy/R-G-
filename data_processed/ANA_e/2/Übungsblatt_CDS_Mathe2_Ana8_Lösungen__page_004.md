## 4. Höhenlinien 

<|ref|>text<|/ref|><|det|>[[114, 98, 841, 117]]<|/det|>
Berechnen und skizzieren Sie für die gegebene Funktion jeweils die Höhenlinien. 

<|ref|>equation<|/ref|><|det|>[[114, 116, 660, 137]]<|/det|>
a) \(f(x, y) = 3x + 6y\) b) \(f(x, y) = \sqrt{y - x^2}\) 

<|ref|>equation<|/ref|><|det|>[[114, 135, 636, 166]]<|/det|>
c) \(f(x, y) = x^2 + y^2 - 2y\) d) \(f(x, y) = \frac{x^2 + y^2}{2y}\)

<|ref|>text<|/ref|><|det|>[[114, 181, 150, 198]]<|/det|>
a) 

<|ref|>text<|/ref|><|det|>[[114, 197, 485, 214]]<|/det|>
Für die Höhenlinie zur Höhe L ergibt sich 

<|ref|>equation<|/ref|><|det|>[[164, 212, 561, 234]]<|/det|>
\[L = f(x; y) = 3x + 6y \quad \text{mit } L \in \mathbb{R}.\]

<|ref|>equation<|/ref|><|det|>[[114, 244, 541, 265]]<|/det|>
\[L - 3x = 6y \quad \text{mit } L \in \mathbb{R}.\]

<|ref|>equation
\[y = \frac{L - 3x}{6} = \frac{L}{6} - \frac{x}{2}.\]

<|ref|>text<|/ref|><|det|>[[114, 310, 415, 327]]<|/det|>
Man erhält somit als Höhenlinien 

<|ref|>equation<|/ref|><|det|>[[120, 330, 370, 366]]<|/det|>
\[y = -\frac{x}{2} + \frac{L}{6} \quad \text{mit } L \in \mathbb{R}.\]

<|ref|>text<|/ref|><|det|>[[114, 640, 150, 657]]<|/det|>
b) 

<|ref|>text<|/ref|><|det|>[[114, 656, 490, 674]]<|/det|>
Für die Höhenlinie zur Höhe L ergibt sich 

<|ref|>equation
\[L = f(x; y) = \sqrt{y - x^2} \quad \text{mit } L \in \mathbb{R}.\]

<|ref|>equation<|/ref|><|det|>[[0, 682, 536, 736]]<|/det|>
\[L^2 = y - x^2 \quad \text{mit } L \in \mathbb{R}.\]

<|ref|>text<|/ref|><|det|>[[114, 735, 415, 753]]<|/det|>
Man erhält somit als Höhenlinien 

<|ref|>equation<|/ref|><|det|>[[0, 750, 370, 775]]<|/det|>
\[y = x^2 + L^2 \quad \text{mit } L \in \mathbb{R}_0^+.\]
