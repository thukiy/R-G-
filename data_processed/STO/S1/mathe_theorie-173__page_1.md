# Wahrscheinlichkeitsverteilung
## einer diskreten Zufallsvariablen 

<|ref|>text<|/ref|><|det|>[[24, 156, 857, 245]]<|/det|>
**Definition:** Die Wahrscheinlichkeitsverteilung einer diskreten Zufallsvariablen X lässt sich durch die Wahrscheinlichkeitsfunktion 

<|ref|>equation<|/ref|><|det|>[[144, 288, 956, 500]]<|/det|>
\[ \text{p}_i = \begin{cases} 0 & \text{übrige } x \\ 1 & \text{oder } x = x_i \end{cases} \quad \text{i} = 1, 2, 3 \dots \quad \text{p}_i: \text{Wahrscheinlichkeit, dass } X \text{ den Wert } x_i \text{ annimmt} \]

<|ref|>sub_title<|/ref|><|det|>[[214, 567, 295, 612]]<|/det|>
## ODER 

<|ref|>text<|/ref|><|det|>[[133, 690, 695, 780]]<|/det|>
durch die Verteilungsfunktion \(F(x) = P(X \le x) = \sum_{x_i \le x} f(x_i)\) beschreiben. 

<|ref|>text<|/ref|><|det|>[[24, 820, 496, 920]]<|/det|>
Eigenschaften: 1. \(f(x_i) \ge 0\)
2. \(f(x)\) ist normiert: \(\sum_{i=1}^{\infty} f(x_i) = 1\)
