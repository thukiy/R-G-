# Differentialgleichungen 

<|ref|>sub_title<|/ref|><|det|>[[44, 123, 145, 163]]<|/det|>
## Definition: 

<|ref|>text<|/ref|><|det|>[[44, 187, 950, 277]]<|/det|>
Eine DGL n-ter Ordnung/n-ten Grades für eine Funktion \(y(x)\) ist eine Gleichung zwischen \(x, y\) und den Ableitungen von \(y\) bis einschliesslich n-ter Ordnung: 

<|ref|>equation<|/ref|><|det|>[[44, 315, 420, 370]]<|/det|>
\[ \text{Implizite Form: } F(x, y, y', y'' \dots y^{(n)}) = 0 \]

<|ref|>equation<|/ref|><|det|>[[44, 404, 464, 460]]<|/det|>
\[ \text{Explizite Form: } y^{(n)} = f(x, y, y', y'' \dots y^{(n-1)}) \]

<|ref|>text<|/ref|><|det|>[[44, 545, 529, 586]]<|/det|>
→ Ordnung/Grad entspricht der höchsten Ableitung. 

<|ref|>text<|/ref|><|det|>[[44, 586, 925, 631]]<|/det|>
→ DGL haben unendlich viele Lösungen, nur mittels AB und RB erhält man eindeutige Lösung.
