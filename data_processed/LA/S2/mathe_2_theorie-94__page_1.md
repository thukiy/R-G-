Def: \(\vec{v}_1 \ldots \vec{v}_n\) heissen linear unabhängig genau dann, wenn 

<|ref|>equation<|/ref|><|det|>[[104, 72, 572, 102]]<|/det|>
\[x_1 \vec{v}_1 + x_2 \vec{v}_2 + \ldots + x_n \vec{v}_n = 0 \quad x_i \in \mathbb{K}\]

<|ref|>text<|/ref|><|det|>[[104, 111, 450, 138]]<|/det|>
nun die Lösung \(x_i = 0\) besitzt. 

<|ref|>text<|/ref|><|det|>[[104, 149, 616, 178]]<|/det|>
Andenfalls heissen \(\vec{v}_1 \ldots \vec{v}_n\) linear abhängig 

<|ref|>text<|/ref|><|det|>[[18, 202, 327, 234]]<|/det|>
Bsp: \(\cdot (1, 1), (1, 1), (1, 1)\) 

<|ref|>equation<|/ref|><|det|>[[150, 241, 480, 272]]<|/det|>
\[x_1 \cdot (1, 1) + x_2 \cdot (1, 1) + x_3 \cdot (1, 1) = 0\]

<|ref|>text<|/ref|><|det|>[[150, 284, 440, 310]]<|/det|>
\(\rightarrow\) nun für \(x_i = 0\) erfüllt 

<|ref|>text<|/ref|><|det|>[[150, 323, 410, 350]]<|/det|>
\(\rightarrow\) linear unabhängig 

<|ref|>text<|/ref|><|det|>[[40, 383, 866, 450]]<|/det|>
\(\rightarrow\) linear unabhängig: Keiner der Veltoren ist als LH der anderen n-1 Veltoren dasselbe

Def: \(B = \{\vec{v}_1 \ldots \vec{v}_n\} \subset V\) heisst genau dann Basis von \(T\), wenn die \(\vec{v}_i\) linear unabhängig sind und ein Erzeugendensystem bilden. 

<|ref|>text<|/ref|><|det|>[[40, 558, 736, 585]]<|/det|>
\(\rightarrow\) jedes \(\vec{w} \in V\) lässt sich als LH der Basis B darstellen 

<|ref|>text<|/ref|><|det|>[[18, 618, 920, 682]]<|/det|>
Satz: V hat eine Basis aus n Veltoren. Dann haben alle Bazen dieses Veltortraums genau n Veltoren. 

<|ref|>text<|/ref|><|det|>[[18, 715, 962, 780]]<|/det|>
Def: Ein Veltortraum mit einer Basis aus n Veltoren besitzt die Dimension n: 

<|ref|>equation<|/ref|><|det|>[[104, 755, 737, 781]]<|/det|>
\[\dim(V) = n \quad \text{für } n \text{ Dimension } n \text{ Bazen nötig}\]

<|ref|>text<|/ref|><|det|>[[40, 814, 850, 840]]<|/det|>
\(\rightarrow\) weniger als n Veltoren können kein Erzeugendensystem bilden 

<|ref|>text<|/ref|><|det|>[[40, 852, 720, 879]]<|/det|>
\(\rightarrow\) es sind nie mehr als n Veltoren linear unabhängig
