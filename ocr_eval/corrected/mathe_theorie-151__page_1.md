# Verknüpfungen von Ereignissen 

<|ref|>text<|/ref|><|det|>[[25, 81, 850, 128]]<|/det|>
Da Ereignisse durch Mengen dargestellt werden, können sie wie diese verknüpft werden.

- Durchschnitt : A ∩ B 

<|ref|>text<|/ref|><|det|>[[42, 255, 548, 315]]<|/det|>
