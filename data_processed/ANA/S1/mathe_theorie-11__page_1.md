# Mengenlehre
## Mengenrelationen 

<|ref|>text<|/ref|><|det|>[[46, 152, 464, 195]]<|/det|>
**A = B**: A und B besitzen dieselben Elemente 

<|ref|>text<|/ref|><|det|>[[46, 240, 949, 285]]<|/det|>
**A ⊆ B**: A ist Teilmenge von B, d. h. jedes Element von A ist auch Element von B; A = B ist möglich 

<|ref|>text<|/ref|><|det|>[[46, 325, 861, 370]]<|/det|>
**A ⊂ B**: A ist echte Teilmenge von B, d. h. B besitzt mehr Elemente als A; {A ⊆ B ∧ A ≠ B} 

<|ref|>text<|/ref|><|det|>[[46, 410, 374, 453]]<|/det|>
**A ∉ B**: A ist keine Teilmenge von B 

<|ref|>equation<|/ref|><|det|>[[46, 503, 199, 592]]<|/det|>
\[
\begin{align*}
A \subseteq B \Leftrightarrow B \supseteq A \\
A \subset B \Leftrightarrow B \supset A
\end{align*}
\]

<|ref|>text<|/ref|><|det|>[[46, 645, 144, 685]]<|/det|>
Beispiele: 

<|ref|>text<|/ref|><|det|>[[46, 688, 230, 732]]<|/det|>
- {1, 2} ⊆ {1, 2, 3} 

<|ref|>text<|/ref|><|det|>[[46, 732, 230, 777]]<|/det|>
- {1, 2} ⊂ {1, 2, 3} 

<|ref|>text<|/ref|><|det|>[[46, 777, 208, 822]]<|/det|>
- {1, 2} ⊆ {1, 2} 

<|ref|>text<|/ref|><|det|>[[46, 822, 206, 867]]<|/det|>
- {1, 2} ∉ {1, 2} 

<|ref|>text<|/ref|><|det|>[[46, 867, 140, 911]]<|/det|>
- Ø ⊆ A 

<|ref|>text<|/ref|><|det|>[[46, 911, 140, 956]]<|/det|>
- Ø ⊂ A
