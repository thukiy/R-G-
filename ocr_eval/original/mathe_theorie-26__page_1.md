# Folgen 

<|ref|>text<|/ref|><|det|>[[30, 85, 981, 175]]<|/det|>

## Definition : 

<|ref|>text<|/ref|><|det|>[[214, 520, 968, 710]]<|/det|>
Eine Folge \(a_n\) von reellen Zahlen ist eine Abbildung
\(f: \mathbb{N} \to \mathbb{R}\)
\(n \to f(n) = a_n\)
die jedem \(n\) eindeutig ein \(a_n\) zuordnet.
\(a_n\) : Folgeglieder 

<|ref|>text<|/ref|><|det|>[[20, 732, 525, 765]]<|/det|>
→ jede Folge hat einen Stattwert 

<|ref|>text<|/ref|><|det|>[[20, 770, 816, 805]]<|/det|>
→ jedes \(a_k\) hat einen eindeutigen Nachfolger \(a_{k+1}\) 

<|ref|>text<|/ref|><|det|>[[123, 809, 475, 842]]<|/det|>
↔ Reihenfolge wichtig
