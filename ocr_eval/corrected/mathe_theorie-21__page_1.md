# Abbildungen 

<|ref|>text<|/ref|><|det|>[[28, 88, 490, 119]]<|/det|>
- grundlegendes Konzept in Mathe 

<|ref|>text<|/ref|><|det|>[[28, 127, 985, 157]]<|/det|>
- einem Element des Menge A wird ein Element des Menge B zugeordnet 

<|ref|>image<|/ref|><|det|>[[280, 160, 456, 210]]<|/det|>
 

<|ref|>text<|/ref|><|det|>[[65, 220, 940, 333]]<|/det|>
Def: Eine Abbildung \(f\) aus einer Menge \(A\) in eine Menge \(B\) ist eine Vorschrift, die jedem \(a \in A\) genau ein
\(b = f(a) \in f(b) \subseteq B\) zuordnet. 

<|ref|>text<|/ref|><|det|>[[65, 360, 380, 392]]<|/det|>
A: Definitionsmenge 

<|ref|>text<|/ref|><|det|>[[65, 400, 408, 432]]<|/det|>
B: Werke - / Zielmenge 

<|ref|>text<|/ref|><|det|>[[65, 437, 350, 470]]<|/det|>
\(f(A)\): Bild (menge) 

<|ref|>equation<|/ref|><|det|>[[160, 476, 666, 526]]<|/det|>
\[ = \{ b \in B \mid \exists a \in A \text{ mit } b = f(a) \} \]

<|ref|>text<|/ref|><|det|>[[65, 536, 644, 567]]<|/det|>
→ Zielmenge B darf grösser als \(f(A)\) sein 

<|ref|>text<|/ref|><|det|>[[65, 575, 870, 625]]<|/det|>
→ Abbildung ist eindeutig definiert durch Definitionsmenge, Zielmenge und Abbildungsvorschrift 

<|ref|>text<|/ref|><|det|>[[65, 633, 440, 704]]<|/det|>
Schreibweise: \(f: A \to B\)
\(a \to f(a)\)
\(b \to f(b)\) 

<|ref|>text<|/ref|><|det|>[[275, 712, 760, 760]]<|/det|>
\(a\): unabhängige Variable, Argument
\(b\): abhängige Variable 

<|ref|>text<|/ref|><|det|>[[20, 787, 493, 821]]<|/det|>
Bsp: \(f: \{1, 2\} \to \{3, 5, 6\}\) 

<|ref|>equation<|/ref|><|det|>[[224, 830, 435, 860]]<|/det|>
\[ x \to f(x) = 3x \]

<|ref|>equation<|/ref|><|det|>[[260, 868, 370, 912]]<|/det|>
\[ \begin{cases} f(1) = 3 \\ f(2) = 6 \end{cases} \]

<|ref|>text<|/ref|><|det|>[[275, 927, 905, 955]]<|/det|>
↔ Wert 5 aus Zielmenge wird nicht angenommen
