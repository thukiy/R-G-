## Hinreichende Bedingungen für lokale Extrem 

<|ref|>text<|/ref|><|det|>[[52, 184, 730, 255]]<|/det|>
f : D ⊆ ℝ → ℝ sei Δ mal stetig differenzierbar
Für a ∈ D gilt : f'(a) = 0. Dann gilt : 

<|ref|>text<|/ref|><|det|>[[78, 262, 736, 296]]<|/det|>
a) f''(a) < 0 ⇒ f hat in a lokales Maximum 

<|ref|>text<|/ref|><|det|>[[78, 301, 715, 335]]<|/det|>
b) f''(a) > 0 ⇒ f hat in a lokales Minimum 

<|ref|>text<|/ref|><|det|>[[78, 340, 930, 376]]<|/det|>
c) f''(a) = 0 und f'''(a) ≠ 0 ⇒ f hat in a Sattelpunkt 

<|ref|>text<|/ref|><|det|>[[17, 401, 960, 472]]<|/det|>
→ falls gilt f''(a) = f'''(a) = 0, müssen erweilte Kriterien betrachtet
werden : es gelte f'(a) = f''(a) = ... = f^{(m-1)}(a) = 0 und f^{(m)} ≠ 0 

<|ref|>text<|/ref|><|det|>[[78, 478, 580, 512]]<|/det|>
a) m sei gerade und f^{(m)}(a) < 0 : 

<|ref|>text<|/ref|><|det|>[[125, 518, 655, 550]]<|/det|>
an (a, f(a)) liegt lokales Maximum vor 

<|ref|>text<|/ref|><|det|>[[78, 555, 585, 589]]<|/det|>
b) m sei gerade und f^{(m)}(a) > 0 : 

<|ref|>text<|/ref|><|det|>[[125, 595, 680, 628]]<|/det|>
an (a, f(a)) liegt lokales Minimum vor 

<|ref|>text<|/ref|><|det|>[[78, 635, 740, 668]]<|/det|>
c) m sei ungeaade : (a, f(a)) ist Sattelpunkt
