## Marszaphen 

<|ref|>text<|/ref|><|det|>[[55, 263, 888, 295]]<|/det|>
Ewartungswert : \(E(x) = \sum x_i \cdot f(x_i) = \mu\) arithmetisches Mittel 

<|ref|>text<|/ref|><|det|>[[55, 320, 770, 348]]<|/det|>
Dineratiblasta : \(E(aX + b) = a \cdot E(X) + b\) a, b ∈ ℝ 

<|ref|>text<|/ref|><|det|>[[55, 375, 576, 408]]<|/det|>
Varianz : \(Var(X) = \sigma^2 = \sum_i (x_i - \mu)^2 \cdot f(x_i)\) 

<|ref|>text<|/ref|><|det|>[[195, 415, 670, 444]]<|/det|>
Man für quadrielle Abweichung von \(\mu\) 

<|ref|>text<|/ref|><|det|>[[55, 472, 456, 503]]<|/det|>
Standardabweichung : \(\sigma = \sqrt{\sigma^2}\) 

<|ref|>text<|/ref|><|det|>[[348, 511, 802, 540]]<|/det|>
durchschnittliche Abweichung von \(\mu\) 

<|ref|>text<|/ref|><|det|>[[55, 567, 686, 599]]<|/det|>
Varianzverschiebungstraa : \(E(X^2) - [E(X)]^2 = \sigma^2\) 

<|ref|>text<|/ref|><|det|>[[55, 625, 518, 655]]<|/det|>
Dineratität : \(Var(aX + b) = a^2 \cdot Var(X)\) 

<|ref|>text<|/ref|><|det|>[[55, 678, 411, 713]]<|/det|>
Standardisierung : \(\bar{z} = \frac{x - \mu}{\sigma}\)
