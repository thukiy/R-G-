<|ref|>text<|/ref|><|det|>[[18, 245, 970, 315]]<|/det|>
Verteilungsfunktion F(x) eine Dupellvariable X gibt die Wahrscheinlichkeit
an, dass X einen Wert \(\leq x \in \mathbb{R}\) annimmt: \(F(x) = P(X \leq x)\) 

<|ref|>text<|/ref|><|det|>[[18, 343, 845, 411]]<|/det|>
Wahrscheinlichkeitsverteilung kann durch Dichtefunktion f(x) oder
Verteilungsfunktion F(x) beschrieben werden: 

<|ref|>equation<|/ref|><|det|>[[103, 415, 512, 455]]<|/det|>
\[f(x) = P(X \leq x) = \int_{-\infty}^{x} f(v) \, dv\]

<|ref|>text<|/ref|><|det|>[[50, 476, 430, 508]]<|/det|>
Eigenschaften: 
- \(f(x) \geq 0\) 
- \(f(x)\) ist normiert: \(\int_{-\infty}^{\infty} f(x) \, dx = 1\) 
- \(F'(x) = f(x)\) 
- Wahrscheinlichkeit, dass X einen Wert zwischen 

<|ref|>text<|/ref|><|det|>[[320, 515, 760, 545]]<|/det|>
a und b annimmt (a \(\leq\) x \(\leq\) b): 

<|ref|>equation<|/ref|><|det|>[[320, 551, 839, 590]]<|/det|>
\[P(a \leq x \leq b) = \int_a^b f(x) \, dx = F(b) - F(a)\]
