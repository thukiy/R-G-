# Mengenlehre
## Mengenoperationen 

<|ref|>text<|/ref|><|det|>[[48, 181, 444, 228]]<|/det|>
**Mengendifferenz:** \(A \setminus B = \{x | x \in A \land x \notin B\}\) 

<|ref|>text<|/ref|><|det|>[[48, 345, 133, 384]]<|/det|>
Beispiel: 

<|ref|>text<|/ref|><|det|>[[48, 375, 265, 480]]<|/det|>
- \(\{1, 2\} \setminus \{2, 3\} = \{\)
- \(\{1, 2\} \setminus \{1, 2\} = \{\}\)

<|ref|>text<|/ref|><|det|>[[48, 585, 660, 633]]<|/det|>
**Symmetrische Differenz:** \(A \Delta B = (A \setminus B) \cup (B \setminus A) = (A \cup B) \setminus (A \cap B)\) 

<|ref|>text<|/ref|><|det|>[[48, 750, 133, 789]]<|/det|>
Beispiel: 

<|ref|>text<|/ref|><|det|>[[48, 780, 320, 925]]<|/det|>
- \(\{1, 2\} \Delta \{2, 3\} = \{\)
- \(\{1, 2\} \Delta \{1, 2\} = \{\}\)
- \(\{1, 2\} \Delta \{3, 4\} = \{\)
- \(\{1, 2\} \Delta \{1, 3, 4\} = \{\}\)
