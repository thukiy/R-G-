# Beschreibende Statistik 

<|ref|>sub_title<|/ref|><|det|>[[26, 85, 189, 135]]<|/det|>
## Grundbegriffe 

<|ref|>text<|/ref|><|det|>[[26, 150, 480, 195]]<|/det|>
Grundgesamtheit/Population/statistische Masse: 

<|ref|>text<|/ref|><|det|>[[26, 196, 611, 240]]<|/det|>
Menge von Objekten, deren Eigenschaften untersucht werden. 

<|ref|>text<|/ref|><|det|>[[26, 270, 765, 315]]<|/det|>
Elemente der Grundgesamtheit heissen Merkmalsträger/Untersuchungseinheit. 

<|ref|>text<|/ref|><|det|>[[26, 345, 123, 386]]<|/det|>
Merkmal: 

<|ref|>text<|/ref|><|det|>[[26, 388, 718, 432]]<|/det|>
Eigenschaft des Merkmalsträgers, das bei Untersuchung von Interesse ist. 

<|ref|>text<|/ref|><|det|>[[26, 463, 290, 505]]<|/det|>
Merkalswert/-ausprägung: 

<|ref|>text<|/ref|><|det|>[[26, 506, 650, 550]]<|/det|>
Wert, den der Merkmalsträger bei Beobachtung/Messung annimmt. 

<|ref|>text<|/ref|><|det|>[[26, 608, 880, 784]]<|/det|>
Gegeben seien n statistische Elemente (n ∈ N). X sei ein Merkmal. xᵢ = X(i), 1 ≤ i ≤ n, bezeichnet die Merkmalsausprägung für das statistische Element i. m ∈ N sei die Anzahl der unterschiedlichen Merkmalsausprägungen, die X auf der Grundgesamtheit annimmt und wir nummerieren die Merkmalsausprägungen mit aᵏ, 1 ≤ k ≤ m. 

<|ref|>text<|/ref|><|det|>[[26, 785, 552, 830]]<|/det|>
Absolute Häufigkeit: hᵏ = Anzahl der xᵢ, die gleich aᵏ sind 

<|ref|>text<|/ref|><|det|>[[26, 830, 290, 901]]<|/det|>
Relative Häufigkeit: fᵏ = hᵏ/n 

<|ref|>equation<|/ref|><|det|>[[360, 840, 670, 951]]<|/det|>
\[ \sum_{k=1}^{m} h_k = n \quad \sum_{k=1}^{m} f_k = 1 \]
