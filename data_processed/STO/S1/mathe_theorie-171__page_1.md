# Verteilungsfunktion 

<|ref|>sub_title<|/ref|><|det|>[[24, 113, 320, 155]]<|/det|>
## Definition: Verteilungsfunktion 

<|ref|>text<|/ref|><|det|>[[130, 155, 872, 290]]<|/det|>
Die Verteilungsfunktion F(x) einer Zufallsvariablen X gibt die Wahrscheinlichkeit an, dass X einen Wert kleiner oder gleich einer reellen Zahl x annimmt: 
\[
F(x) = P(X \le x).
\] 

<|ref|>text<|/ref|><|det|>[[131, 345, 531, 388]]<|/det|>
X wird durch F(X) vollständig beschrieben. 

<|ref|>text<|/ref|><|det|>[[24, 475, 583, 685]]<|/det|>
Eigenschaften: 1. F(x) ist monoton wachsend: \(0 \le F(x) \le 1\).
2. \(\lim_{x \to -\infty} F(x) = 0\): unmögliches Ereignis
3. \(\lim_{x \to \infty} F(x) = 1\): sicheres Ereignis
4. \(P(a < X \le b) := F(b) - F(a)\) 

<|ref|>text<|/ref|><|det|>[[24, 746, 606, 789]]<|/det|>
Folgende Eigenschaften von X sind von generellem Interesse: 

<|ref|>text<|/ref|><|det|>[[24, 789, 303, 832]]<|/det|>
a) Wertebereich (X: Ω → ℝ) 

<|ref|>text<|/ref|><|det|>[[24, 832, 880, 921]]<|/det|>
b) Wahrscheinlichkeit P, dass X einen bestimmten Wert annimmt (diskrete Variable) bzw. in einem bestimmten Intervall liegt (stetige Variable)
