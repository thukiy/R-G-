# Summenzeichen : Σ 

<|ref|>text<|/ref|><|det|>[[60, 69, 867, 139]]<|/det|>
verwendet, um eine Summe mit gleichartigen Summanden
kann aufschreiben 

<|ref|>equation<|/ref|><|det|>[[96, 137, 520, 186]]<|/det|>
\[ \sum_{k=1}^{n} a_k := a_1 + a_2 + a_3 + \dots + a_n \]

<|ref|>text<|/ref|><|det|>[[100, 204, 715, 234]]<|/det|>
k : dauervariabele, durchlauf weite von 1...n 

<|ref|>text<|/ref|><|det|>[[116, 245, 620, 273]]<|/det|>
→ dauervariabele kann geändert werden 

<|ref|>equation<|/ref|><|det|>[[175, 275, 404, 322]]<|/det|>
\[ \sum_{k=1}^{n} a_k = \sum_{m=1}^{n} a_m \]

<|ref|>text<|/ref|><|det|>[[116, 343, 750, 373]]<|/det|>
→ Indexverschiebung kann durchgeführt werden 

<|ref|>equation<|/ref|><|det|>[[210, 372, 450, 419]]<|/det|>
\[ \sum_{k=1}^{n} a_k = \sum_{k=1+n}^{n+s} a_{k-s} \]

<|ref|>text<|/ref|><|det|>[[212, 439, 978, 469]]<|/det|>
↪ hilfreich beim Zusammenfassen von Δ oder mehr Summen 

<|ref|>text<|/ref|><|det|>[[116, 488, 718, 530]]<|/det|>
→ Regeln : 

<|ref|>equation<|/ref|><|det|>[[210, 488, 717, 599]]<|/det|>
\[ \begin{align*} \circ \sum_{k=1}^{n} (a_k + b_k) &= \sum_{k=1}^{n} a_k + \sum_{k=1}^{n} b_k \\ \circ \sum_{k=1}^{n} (c \cdot a_k) &= c \cdot \sum_{k=1}^{n} a_k \end{align*} \]

<|ref|>text<|/ref|><|det|>[[25, 612, 590, 655]]<|/det|>
Bsp : 

<|ref|>equation<|/ref|><|det|>[[150, 610, 592, 660]]<|/det|>
\[ \circ \sum_{k=1}^{5} k^2 = 1^2 + 2^2 + 3^2 + 4^2 + 5^2 = 55 \]

<|ref|>equation<|/ref|><|det|>[[150, 666, 805, 712]]<|/det|>
\[ \circ \sum_{k=m}^{n} C = C \cdot \sum_{k=m}^{n} 1 = C \cdot (n-m+1) \quad C \in \mathbb{R} \]

<|ref|>equation<|/ref|><|det|>[[150, 722, 633, 770]]<|/det|>
\[ \circ \sum_{k=1}^{5} \frac{(-1)^k}{2^k} = \frac{-1}{2} + \frac{1}{4} + \frac{-1}{8} + \frac{1}{16} + \frac{-1}{32} \]

<|ref|>text<|/ref|><|det|>[[150, 780, 723, 825]]<|/det|>
\[ \circ \sum_{k=5}^{8} a_k = \sum_{k=0}^{3} a_{k+5} \quad \text{→ Indexverschiebung} \]

<|ref|>equation<|/ref|><|det|>[[150, 836, 416, 884]]<|/det|>
\[ \circ \sum_{k=4}^{40} (k+4)^2 = \sum_{k=8}^{44} k^2 \]

<|ref|>equation<|/ref|><|det|>[[150, 896, 480, 944]]<|/det|>
\[ \circ \sum_{k=1}^{n} a_k = \sum_{k=1}^{n} a_{n+1-k} \]
