<|ref|>text<|/ref|><|det|>[[36, 361, 580, 390]]<|/det|>
→ α ist unabhängig von der Wahl von r 

<|ref|>text<|/ref|><|det|>[[36, 400, 844, 430]]<|/det|>
→ α hat keine Masseinheit, oftmals Radiant (rad) verwendet 

<|ref|>text<|/ref|><|det|>[[36, 435, 585, 472]]<|/det|>
→ α voll = \(\frac{b}{r}\) = \(\frac{u}{r}\) \(\frac{2\pi r}{r}\) = 2π (Vollkreis) 

<|ref|>text<|/ref|><|det|>[[78, 476, 595, 512]]<|/det|>
α halb = \(\frac{b}{r}\) = \(\frac{\pi r}{r}\) = π (Halbkreis) 

<|ref|>text<|/ref|><|det|>[[36, 519, 857, 550]]<|/det|>
→ Winkel im Bogenmass oftmals im Vielfachen von π angegeben
