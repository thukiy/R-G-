a) 

<|ref|>text<|/ref|><|det|>[[114, 115, 667, 134]]<|/det|>
Mit der Kettenregel ergeben sich die partiellen Ableitungen: 

<|ref|>equation<|/ref|><|det|>[[128, 144, 767, 186]]<|/det|>
\[f_x(x, y) = \frac{2 + 3y}{2\sqrt{2x + 3xy + 4y}} = \frac{1}{\sqrt{2x + 3xy + 4y}} + \frac{3}{2}\frac{y}{\sqrt{2x + 3xy + 4y}},\]

<|ref|>equation<|/ref|><|det|>[[128, 198, 765, 239]]<|/det|>
\[f_y(x, y) = \frac{3x + 4}{2\sqrt{2x + 3xy + 4y}} = \frac{3}{2}\frac{x}{\sqrt{2x + 3xy + 4y}} + \frac{2}{\sqrt{2x + 3xy + 4y}}.\]

<|ref|>text<|/ref|><|det|>[[114, 258, 386, 276]]<|/det|>
Daraus resultieren die Werte 

<|ref|>equation<|/ref|><|det|>[[302, 284, 590, 320]]<|/det|>
\[f_x(1, 1) = \frac{5}{6} \text{ und } f_y(1, 1) = \frac{7}{6}.\]

<|ref|>equation<|/ref|><|det|>[[270, 410, 620, 428]]<|/det|>

<|ref|>text<|/ref|><|det|>[[114, 430, 386, 448]]<|/det|>

<|ref|>equation<|/ref|><|det|>[[272, 464, 621, 483]]<|/det|>

c)

<|ref|>equation<|/ref|><|det|>[[278, 539, 475, 559]]<|/det|>

<|ref|>equation<|/ref|><|det|>[[278, 571, 625, 591]]<|/det|>

<|ref|>equation<|/ref|><|det|>[[295, 650, 606, 669]]<|/det|>

<|ref|>text<|/ref|><|det|>[[120, 690, 479, 707]]<|/det|>
Differenziert wird mit Hilfe der Kettenregel: 

<|ref|>equation<|/ref|><|det|>[[174, 717, 844, 763]]<|/det|>
\[z = \underbrace{(2x - 3y^2)^5}_{u} = u^5 \text{ mit } u = 2x - 3y^2 \text{ und } \frac{\partial u}{\partial x} = 2, \quad \frac{\partial u}{\partial y} = -6y\]

<|ref|>equation<|/ref|><|det|>[[174, 778, 694, 816]]<|/det|>
\[z_x = \frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \cdot \frac{\partial u}{\partial x} = 5u^4 \cdot 2 = 10u^4 = 10(2x - 3y^2)^4\]

<|ref|>equation<|/ref|><|det|>[[174, 827, 797, 864]]<|/det|>
\[z_y = \frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} \cdot \frac{\partial u}{\gamma} = 5u^4 \cdot (-6y) = -30yu^4 = -30y(2x - 3y^2)^4\]
