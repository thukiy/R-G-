# Funktionseigenschaften 

<|ref|>text<|/ref|><|det|>[[52, 88, 355, 118]]<|/det|>
\(f(x)\) sei reelle Funktion 

<|ref|>sub_title<|/ref|><|det|>[[21, 147, 175, 175]]<|/det|>
## Symmetrie 

<|ref|>text<|/ref|><|det|>[[52, 187, 916, 215]]<|/det|>
- Symmetrisch aus y-Achse bzw. gerade bzw. positive Parität : 
\[f(-x) = f(x)\] 

<|ref|>text<|/ref|><|det|>[[52, 268, 933, 295]]<|/det|>
- punktsymmetrisch aus Wsprüng bzw. ungerade bzw. negative 

<|ref|>text<|/ref|><|det|>[[52, 300, 470, 334]]<|/det|>
Parität : \(f(-x) = -f(x)\) 

<|ref|>text<|/ref|><|det|>[[21, 360, 300, 392]]<|/det|>
**Bsp** : 
- \(f(x) = x^2\) 

<|ref|>equation<|/ref|><|det|>[[161, 400, 620, 430]]<|/det|>
\[f(-x) = (-x)^2 = x^2 \quad \text{→ gerade}\]

<|ref|>equation<|/ref|><|det|>[[161, 437, 680, 468]]<|/det|>
- \(f(x) = x^3\) 

<|ref|>equation<|/ref|><|det|>[[161, 475, 688, 508]]<|/det|>
\[f(-x) = (-x)^3 = -x^3 \quad \text{→ ungerade}\]

<|ref|>sub_title<|/ref|><|det|>[[21, 536, 216, 567]]<|/det|>
## Verschiebung 

<|ref|>text<|/ref|><|det|>[[52, 576, 707, 644]]<|/det|>
- entlang der x-Richtung um \(\Delta x\), \(\Delta x > 0\) : 
nach rechts (annehmende x-Werte) : 
\[g(x) = f(x - \Delta x)\] 

<|ref|>equation<|/ref|><|det|>[[110, 653, 360, 685]]<|/det|>
nach links : \(g(x) = f(x + \Delta x)\) 

<|ref|>text<|/ref|><|det|>[[52, 688, 500, 720]]<|/det|>
- entlang der y-Richtung um \(\Delta y\), \(\Delta y > 0\) : 
nach oben : \(g(x) = f(x) + \Delta y\) 

<|ref|>equation<|/ref|><|det|>[[110, 728, 512, 760]]<|/det|>
nach unten : \(g(x) = f(x) - \Delta y\) 

<|ref|>text<|/ref|><|det|>[[21, 867, 300, 900]]<|/det|>
**Bsp** : 
- \(f(x) = x^2\)
