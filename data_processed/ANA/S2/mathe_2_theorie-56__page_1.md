# Laplace - Operator : 

<|ref|>equation<|/ref|><|det|>[[50, 52, 192, 80]]<|/det|>
\[f : \mathbb{R}^n \to \mathbb{R}\]

<|ref|>equation<|/ref|><|det|>[[50, 88, 315, 120]]<|/det|>
\[\Delta f = \frac{\partial^2 f}{\partial x_1^2} + \frac{\partial^2 f}{\partial x_2^2} + \dots + \frac{\partial^2 f}{\partial x_n^2}\]

## Rechenregeln : 

<|ref|>text<|/ref|><|det|>[[41, 190, 245, 214]]<|/det|>
- \(\text{div}(\nabla f) = \Delta f\) 

<|ref|>text<|/ref|><|det|>[[41, 228, 247, 252]]<|/det|>
- \(\text{div}(\text{rot } \nabla) = 0\) 

<|ref|>text<|/ref|><|det|>[[41, 266, 220, 290]]<|/det|>
- \(\text{rot}(\nabla f) = 0\)
